Pour afficher des icônes ou des emojis dans vos alertes JavaScript, vous pouvez simplement inclure les emojis directement dans la chaîne de caractères que vous passez à la fonction alert(). Voici comment vous pouvez procéder :
Syntaxe pour utiliser des icônes dans les alertes
La syntaxe de base pour une alerte est la suivante :

javascript
alert('Votre message ici');

Exemple d'utilisation avec des emojis
Pour ajouter des emojis ou des icônes, vous pouvez les inclure directement dans le message. Par exemple :

javascript
alert('🚀 La fusée est prête au décollage !');
alert('⚠️ Attention : ceci est un avertissement !');
alert('✅ Opération réussie !');

Utilisation de codes Unicode
Si vous préférez utiliser les codes Unicode pour les emojis, voici un exemple :

javascript
alert(String.fromCodePoint(0x1F680) + ' La fusée est prête au décollage !'); // 🚀
alert(String.fromCodePoint(0x26A0) + ' Attention : ceci est un avertissement !'); // ⚠️
alert(String.fromCodePoint(0x2705) + ' Opération réussie !'); // ✅

Exemple complet
Voici un exemple complet qui montre comment utiliser des alertes avec des emojis :

javascript
// Afficher une alerte avec un emoji de fusée
alert('🚀 La fusée est prête au décollage !');

// Afficher une alerte avec un emoji d'avertissement
alert('⚠️ Attention : ceci est un avertissement !');

// Afficher une alerte avec un emoji de succès
alert('✅ Opération réussie !');

// Afficher une alerte avec un emoji de chargement
alert('⌛ Chargement en cours...');

Conclusion
En résumé, pour ajouter des icônes ou des emojis dans vos alertes JavaScript, 
il suffit de les inclure directement dans la chaîne passée à alert(). 
Vous pouvez utiliser soit les emojis directement, soit leurs codes Unicode via String.fromCodePoint(). 
Cela rendra vos alertes plus engageantes et visuellement attrayantes.
